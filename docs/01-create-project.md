# Create Spring Boot project

- Go to [Spring Initializr](https://start.spring.io)
- Select details:
  - Project: Maven
  - Language: Java
  - Spring Boot: 3.4.2
  - Java: 17
  - Packaging: Jar
- Initial Dependencies:
  - Spring Web
  - Lombok
  - H2 Database
  - Spring Data JPA
  - Validation
- Press on "Generate" button to download project.
